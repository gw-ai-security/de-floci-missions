# Mission Workspaces

Die eigentlichen Missionsressourcen werden bewusst nicht vorprovisioniert. Lege für eine begonnene Mission einen Ordner nach dem Muster `M00-foundation` an und versioniere darin ausschließlich die selbst erarbeiteten Artefakte.

Die verbindlichen Aufgaben und Meilensteine stehen in `content/DEA-C01_Floci_Hands-on_Missionenplan.md`; der Fortschritt wird über die Web-UI auf Port 3000 verwaltet.
